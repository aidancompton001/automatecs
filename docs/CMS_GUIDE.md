# Anleitung: Inhalte bearbeiten — Automatecs CMS

## Zugang

Öffnen Sie den Content Manager unter:
**https://[ihre-domain]/admin/**

Melden Sie sich mit Ihrem GitHub-Konto an.

---

## 1. Text auf der Startseite ändern

1. Klicken Sie links auf **„Seiten"**
2. Wählen Sie **„Startseite"**
3. Ändern Sie die **Überschrift**, den **Untertitel** oder die **Textabschnitte**
4. Klicken Sie oben auf **„Speichern"**
5. Die Änderung wird automatisch auf der Website veröffentlicht

---

## 2. Hero-Text einer Kategorie ändern

1. Klicken Sie links auf **„Seiten"**
2. Wählen Sie die gewünschte Kategorie (z.B. **„Snackautomaten"**)
3. Ändern Sie **Überschrift** und **Untertitel** im Hero-Bereich
4. Klicken Sie auf **„Speichern"**

---

## 3. Einen Blog-Beitrag hinzufügen

1. Klicken Sie links auf **„Blog"**
2. Wählen Sie **„Blog-Beiträge"**
3. Klicken Sie auf **„Neuen Eintrag hinzufügen"** in der Liste
4. Füllen Sie aus:
   - **URL-Slug**: z.B. `mein-neuer-beitrag` (nur Kleinbuchstaben und Bindestriche)
   - **Titel**: Der Titel des Beitrags
   - **Datum**: Das Veröffentlichungsdatum
   - **Kurztext**: Eine kurze Zusammenfassung (wird in der Übersicht angezeigt)
   - **Bild**: Laden Sie ein Bild hoch (wird oben im Beitrag angezeigt)
   - **Inhalt**: Fügen Sie Textabsätze hinzu
5. Klicken Sie auf **„Speichern"**

---

## 4. Impressum oder Datenschutz ändern

1. Klicken Sie links auf **„Rechtliches"**
2. Wählen Sie **„Impressum"** oder **„Datenschutzerklärung"**
3. Bearbeiten Sie den Text im Markdown-Editor
4. Klicken Sie auf **„Speichern"**

---

## 5. Ein Bild ersetzen

1. Gehen Sie zur gewünschten Seite oder zum Blog-Beitrag
2. Klicken Sie auf das Bild-Feld
3. Wählen Sie **„Bild hochladen"** und laden Sie das neue Bild hoch
4. Das alte Bild wird automatisch ersetzt

**Empfohlene Bildgrößen:**
- Hero-Bilder: 1920 x 800 Pixel (max. 200 KB)
- Produkt-Bilder: 800 x 600 Pixel (max. 100 KB)
- Blog-Bilder: 1200 x 630 Pixel (max. 150 KB)
- Format: WebP oder JPEG

---

## Wichtige Hinweise

- Änderungen werden erst nach dem **Speichern** veröffentlicht
- Nach dem Speichern dauert es ca. 1–2 Minuten, bis die Änderung auf der Website sichtbar ist
- **Produkte, Preise und Optionen** können nur vom Entwickler-Team geändert werden
